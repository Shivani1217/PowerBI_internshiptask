## PowerBI_internshiptask
#Global Superstore Sales & Profit Analysis Dashboard

This project analyzes retail sales data using Power BI to identify business insights such as sales performance, profitability, and regional trends.

#Tools Used
- Power BI
- Power Query
- DAX

#Key Insights
• Total Sales and Profit analysis
• Sales by Category and Region
• Profitability comparison
• Discount vs Profit analysis
